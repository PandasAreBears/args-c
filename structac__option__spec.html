Detailed Description

Encapsulates an option specification.

In args-c, an 'option' is a named value. For example, the input argument to unix "sed -i <file>".
Options must have a 'long name' which will be used on the command line as –name.
Optionally, a short name may also be specified and the user can use either interchangeably.
